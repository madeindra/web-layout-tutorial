Flexbox

All red items are 200px x 50px

Parent's display set to flex & flex-direction set to row.
When items are wider than the parent, they will fit in one row.
Parent's display set to flex, flex-direction set to row, flex-wrap set to wrap. This equals to flex-flow set to row wrap
With wrap, when items are wider than the parent, they will not fit, but will be shown in other row.
Parent's display set to flex & flex-direction set to column
When items are wider than the parent, they will fit in one column.
Parent's display set to flex, flex-direction set to column, flex-wrap set to wrap. This equals to flex-flow set to column wrap
With wrap, when items are wider than the parent, they will not fit, but will be shown in other column.
